# Birds-Image-Gallery

This repository contains my Birds Image Gallery created using HTML and CSS as part of my web development learning journey.

This project is a simple Birds Gallery webpage created using only HTML and basic CSS. The page displays a collection of bird images arranged in a clean grid layout with a heading and simple header section.

The main goal of this project is to practice core front-end fundamentals such as image alignment, layout spacing, borders, and basic styling without using any frameworks, libraries, or UI/UX tools.

# Features
Image gallery grid layout

Simple header and page title

Clean and minimal styling

Built using only HTML and CSS

No frameworks or UI libraries used

# Purpose

This project was built to strengthen HTML structure and basic CSS layout skills by creating a simple real-world image gallery webpage.
